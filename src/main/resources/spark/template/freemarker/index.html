Secret Santa

Requirements:

List of recipients should consist either list of email addresses (one per row) or list of recipients in the following format: Email address,Username

Examples:

john1@doe.com

john2@doe.com

john3@doe.com

john4@doe.com

or

john1@doe.com,John Doe1

john2@doe.com,John Doe2

john3@doe.com,John Doe3

john4@doe.com,John Doe4


Notes:

System doesn't check whether provided email addresses are valid

Quantity of users should be even number, otherwise system will drop the last user from the list
